## Run Locally

Clone the project

```bash
  git clone https://github.com/KrystofKlen/Chess.git
```

Go to the project directory

```bash
  cd ChessEngine
```

Install Ncurses

```bash
  sudo apt install libncurses5-dev
```

Compile

```bash
    make compile
```

Run
```bash
    ./game
```

## Documentation
